# Creator Insights AI

AI-powered creator analytics platform that compares YouTube and Instagram videos using transcript analysis, engagement metrics, vector search, and Retrieval-Augmented Generation (RAG).

## Features

- YouTube Metadata Extraction
- YouTube Transcript Extraction
- Instagram Reel Analysis (In Progress)
- Gemini Integration
- FastAPI Backend
- LangChain RAG (Upcoming)
- ChromaDB Vector Database (Upcoming)
- Streaming Responses (Upcoming)
- Conversation Memory (Upcoming)

## Tech Stack
